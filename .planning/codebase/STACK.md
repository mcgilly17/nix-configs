## Configuration

**Environment:**
- `.env` files: Not detected in codebase (environment configuration via nix modules)
- Secrets: Managed via sops-nix with age encryption
  - System secrets: `inputs.nix-secrets + "/secrets.yaml"` (private repo)
  - User age key: Derived from SSH key via ssh-to-age

**Build Configuration:**
- flake.nix - Main entry point for all configurations
- devenv.nix - Development shell environment
- devenv.yaml - devenv schema and configuration
- disks.nix files - Disk partitioning per host (ganon, glados, rk1 nodes)
- hardware configuration - Generated per-host (omitted from repo, generated by hardware scan)

**Nix Modules:**
- `modules/common/` - Cross-platform core (core.nix, host-spec.nix)
- `modules/nixos/` - NixOS-specific modules (common.nix, sops.nix, tailscale.nix, apps/)
- `modules/darwin/` - macOS-specific modules (apps/, wm/, system.nix)
- `hosts/nixos/` - Host-specific configurations (ganon, glados, rk1, wsl)
- `users/michael/` - User home-manager configuration (common, darwin, nixos)

## Platform Requirements

**Development:**
- Nix - Flake-capable Nix version (for `nix flake` commands)
- Git - For cloning and managing repos
- SSH key - For accessing nix-secrets private repo

**Production Hosts:**
- **ganon**: x86_64-linux, NVIDIA RTX 30 series+ GPU, GRUB bootloader, Hyprland/Wayland
- **glados**: x86_64-linux (standard server configuration)
- **bowser**: aarch64-darwin (Apple Silicon Mac)
- **sephiroth**: aarch64-linux RK1 dev node with Tailscale
- **zenith-1/2/3**: aarch64-linux RK3588 K3s cluster nodes, Turing Pi 2 form factor
- **ocelot/mantis**: x86_64-linux WSL2 hosts with NVIDIA GPU support

**Hardware-Specific:**
- RK1 nodes: ARM64 Rockchip RK3588, U-Boot/extlinux bootloader, iSCSI support
- WSL Nodes: Windows Subsystem for Linux 2 (WSL2) backend
- Gaming PC (ganon): Requires `nvidia_drm.fbdev=1` kernel parameter
