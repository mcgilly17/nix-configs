The module function signature: `{ lib, pkgs, ... }:` (no guard needed — parent aggregator handles isWSL).

Build `komorebiConfig` attrset in a `let` block with these exact fields:

1. **Schema and paths (KOMO-01):**
   - `"$schema"` = `"https://raw.githubusercontent.com/LGUG2Z/komorebi/v0.1.40/schema.json"` (quoted key — `$` requires string key in Nix)
   - `app_specific_configuration_path` = `"$Env:KOMOREBI_CONFIG_HOME/applications.json"`
   - `window_hiding_behaviour` = `"Cloak"` (Windows 11 only — both hosts are Win11)
   - `cross_monitor_move_behaviour` = `"Insert"`

2. **Layout and spacing (KOMO-01):**
   - `default_workspace_padding` = `10`
   - `default_container_padding` = `10`
   - `border` = `true`
   - `border_width` = `8`
   - `border_offset` = `-1`

3. **Catppuccin Mocha theme (KOMO-03):**
   ```nix
   theme = {
     palette = "Catppuccin";
     name = "Mocha";
     bar_accent = "Sapphire";
     single_border = "Sapphire";
     stack_border = "Green";
     floating_border = "Yellow";
     monocle_border = "Mauve";
     unfocused_border = "Surface1";
   };
   ```

4. **Float rules in komorebi.json, NOT applications.json (KOMO-02):**
   ```nix
   floating_applications = [
     { kind = "Exe"; id = "SystemSettings.exe"; matching_strategy = "Equals"; }
     { kind = "Class"; id = "TaskManagerWindow"; matching_strategy = "Legacy"; }
     { kind = "Class"; id = "OperationStatusWindow"; matching_strategy = "Legacy"; }
     { kind = "Title"; id = "Control Panel"; matching_strategy = "Equals"; }
   ];
   ```

5. **Per-monitor workspaces (KOMO-04):**
   ```nix
   monitors = [
     {
       workspaces = [
         { name = "I";   layout = "BSP"; }
         { name = "II";  layout = "VerticalStack"; }
         { name = "III"; layout = "HorizontalStack"; }
         { name = "IV";  layout = "Grid"; }
         { name = "V";   layout = "Monocle"; }
       ];
     }
     {
       workspaces = [
         { name = "1"; layout = "BSP"; }
         { name = "2"; layout = "VerticalStack"; }
         { name = "3"; layout = "HorizontalStack"; }
       ];
     }
   ];
   ```

6. **Registration:** In the body, set:
   ```nix

     pkgs.writeText "komorebi.json" (builtins.toJSON komorebiConfig);
   ```
